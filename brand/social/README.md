## Exporting to PNG/JPG for upload
Social platforms want raster, so export each SVG:
- **Easiest:** open the `.svg` in a browser (fonts load automatically), then screenshot / "Save as image", or use a browser-based SVG→PNG converter.
- **Figma/Inkscape:** import the SVG and export at the exact pixel dimensions above.
- **Note on fonts:** these banners use Figtree + JetBrains Mono via web import, so export somewhere with
  internet (browser/Figma). For a fully self-contained file, outline the text to paths first.

## Keep it consistent
- Same avatar across every channel.
- Colors and type per `../BRAND.md` — don't recolor the tile or swap the font.
- The mark alone (no wordmark) is fine anywhere the space is tight.
