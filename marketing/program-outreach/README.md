## The 30-second strategy
The whole US AA market is ~14 programs / a few hundred new SAAs per year. You can reach **every
program by name.** Don't scale — personalize. You're a practicing CAA emailing faculty peer-to-peer;
that credibility is the entire wedge. Lead with **first-time NCCAA CERT pass rates** (what programs
are institutionally measured on) and offer a **free pilot** so there's zero risk to saying yes.

## How to run it (≈1 hour/week)
1. **Fill in contacts.** For each program in the CSV, get the program director's name + email from
   the program website (links in the CSV), the AAAEP directory (aaaep.org), or LinkedIn. Mark
   `Director Name`/`Director Email` and clear the `VERIFY` flag. **Never send to an unverified name.**
2. **Prioritize.** Start with `Priority = High`: Georgia programs (Emory, South University — your home
   turf, possible in-person) and the big multi-campus volume programs (Nova, Case Western).
3. **Send Touch 1** from your real name + a macprep.org address (not a generic info@). Personalize the
   one bracketed line per program. Log the date in `Last Contact` and set `Status = Contacted`.
4. **Follow up on schedule.** Touch 2 at ~5–7 days (attach the one-pager PDF). Touch 3 at ~7 days
   after that. Update `Status` as replies come in (Replied / Meeting / Pilot / Won / Pass).
5. **Convert.** Goal of every thread = a 20-min call → free pilot for one current cohort. First
   adopter becomes your reference for the other 13.

## Status legend
`Not started` → `Contacted` → `Replied` → `Meeting set` → `Pilot` → `Won` / `Passed`

## Don't
- Don't blast a BCC list. Send individually; faculty talk to each other.
- Don't fabricate a director name. Verify first.
- Don't lead with features. Lead with pass rates + a teaching tool + zero-risk pilot.
